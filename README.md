# Python-Pandas
## Installation
This program assumes that you have both python 3.14 and relevant python packages installed.
Python can be found on their website, the following code can be used for windows
```
winget install python
```
and Arch Linux Machines.
```
sudo pacman -S python
```

The python pip packages will be installed via the python pip shell:
```
pip install pandas numpy mathplotlib
```
